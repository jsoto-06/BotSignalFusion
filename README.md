# 🚀 SignalFusion Pro - Bitget Trading Bot

**SignalFusion Pro** es una aplicación de Android avanzada diseñada para el trading automatizado en el mercado de **Bitget USDT-Futures**. Utiliza un motor de análisis técnico en tiempo real para ejecutar estrategias basadas en indicadores de precisión.

!

## ✨ Características Principales
* **Interfaz Terminal Pro**: Diseño oscuro de alta visibilidad con indicadores de neón.
* **Motor Multi-Estrategia**: Permite alternar entre 3 modos de trading (Moderado, Agresivo y Breakout).
* **Monitoreo 24/7**: Funciona en segundo plano mediante un Servicio de Primer Plano (Foreground Service).
* **Seguridad Local**: Las API Keys se almacenan de forma segura en el dispositivo, nunca se envían a servidores externos.
* **Cálculo de Riesgo Automático**: Ajusta el tamaño de la posición según la volatilidad del mercado (ATR).

## 📊 Estrategias Incluidas

| Estrategia | Perfil | Objetivo de Trades | Perfil de Riesgo |
| :--- | :--- | :--- | :--- |
| **Moderada** 🟢 | Balanceada | 2-5 trades/día | Medio |
| **Agresiva** 🔴 | Alta Frecuencia | 5-15 trades/día | Alto |
| **Breakout** 🔵 | Francotirador | 1-3 trades/día | Bajo |

## 🛠️ Requisitos e Instalación
1.  **Android Studio** Jellyfish o superior.
2.  **SDK de Android**: Nivel 24 o superior.
3.  **API Keys de Bitget**: Con permisos de lectura y trading en Futuros.

### Configuración Rápida
1.  Clona el repositorio.
2.  Compila la APK en tu dispositivo.
3.  Ingresa tus llaves en la pestaña **Ajustes**.
4.  Configura tu capital (ej: $1000) y apalancamiento.
5.  ¡Enciende el motor en el Dashboard!
